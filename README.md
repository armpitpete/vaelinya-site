# Vaelinya Site

Public static website for `vaelinya.uk`.

## Current direction

Vaelinya is your next adventure.

The site is built as a colourful, illustrated, learnable public doorway into Vaelinya, with four equal homepage paths:

- Read
- World
- Language
- Artefacts

## Build

```bash
npm install
npm run build
```

Cloudflare Pages build settings:

```text
Build command: npm run build
Build output directory: dist
Production branch: main
```
